# Welcome to the Knowledge Base

This is the first post in our new knowledge base and blog. We're using a stack that makes writing a joy:

<!-- more -->

## The Stack

| Tool | Role |
|---|---|
| [MkDocs](https://www.mkdocs.org/) | Static site generator |
| [mkdocs-shadcn](https://github.com/asiffer/mkdocs-shadcn) | shadcn/ui-inspired theme |
| [MkDocs Publisher](https://mkdocs-publisher.github.io/) | Blog engine + Obsidian integration |
| [Obsidian](https://obsidian.md/) | Writing interface (vault = `docs/`) |

## Writing Workflow

Open the `docs/` folder as an Obsidian vault. You can:

- Use `[[WikiLinks]]` to link between notes — they resolve automatically at build time.
- Use Obsidian callouts like `> [!warning]` — converted to admonitions.
- Use the `blog-post` template (in `_templates/`) for consistent frontmatter.

## Publishing

Set `publish: true` in any document's frontmatter to make it appear in the built site. Leave it as `false` (or omit it) to keep it as a draft.

> [!tip]
> You can open this entire repository in Obsidian. Just point the vault at the `docs/` directory.
