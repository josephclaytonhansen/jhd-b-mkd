# Welcome to My Knowledge Base

This is a personal knowledge base and blog built with [MkDocs](https://www.mkdocs.org), the [shadcn theme](https://github.com/asiffer/mkdocs-shadcn), and [MkDocs Publisher](https://mkdocs-publisher.github.io/).

It uses [Obsidian](https://obsidian.md/) as the editing interface — open the `docs/` folder as an Obsidian vault.

## Features

- **Blog** — timestamped posts with categories, tags, and archives
- **WikiLinks** — `[[Page Name]]` links resolved automatically
- **Callouts** — Obsidian-style callouts converted to admonitions
- **Backlinks** — automatic cross-reference discovery
- **Figure captions** — images with `Figure:` captions become proper `<figure>` elements
- **Link markers** — external link ⧉ and email ✉ indicators
