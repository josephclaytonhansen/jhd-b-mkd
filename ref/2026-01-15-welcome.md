# Welcome to the Knowledge Base

This is the first post in our new blog. The stack is simple and fast:

<!-- more -->

## The Stack

| Tool | Role |
|---|---|
| [MkDocs](https://www.mkdocs.org/) | Static site generator |
| [mkdocs-shadcn](https://github.com/asiffer/mkdocs-shadcn) | shadcn/ui-inspired theme |
| [MkDocs Publisher](https://mkdocs-publisher.github.io/) | Blog engine and Obsidian integration |
| [Obsidian](https://obsidian.md/) | Writing interface (vault = `docs/`) |

## Writing Workflow

Open the `docs/` folder as an Obsidian vault. Set `publish: true` in any file to include it in the built site.

> [!tip]
> Use the `blog-post` template in `_templates/` to get consistent frontmatter for new posts.
